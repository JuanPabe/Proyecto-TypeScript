# Proyecto-TypeScript
Proyecto realizado para la materia Ingenieria de Software.
El programa simula la gestion de productos de una tienda de ropa usando TypeScript.

## Tecnologias usadas

- TypeScript
- Node.js
- GitHub
- Integracion continua con GitHub Actions
  
Conceptos aplicados:
- Programación Orientada a Objetos
- Clases abstractas
- Herencia
- Interfaces
- Sobrecarga de métodos
- Ordenamiento de datos
